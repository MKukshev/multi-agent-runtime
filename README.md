# Multi-agent Runtime (Variant B)

Этот репозиторий содержит реализацию нового persistent рантайма для SGR-агентов (Variant B). Проект следует архитектурному плану из каталога `docs/` и включает OpenAI-совместимый gateway, пул долговечных инстансов агентов, каталог инструментов и сервис поиска инструментов.

## Структура
- `docs/` — дорожная карта и подробный план миграции с текущего SGR-Agent.
- `src/platform/` — исходники нового рантайма (скелет пакетов):
  - `core/` — переносимые агенты, инструменты, SGR builder и поток SSE.
  - `gateway/` — OpenAI-совместимые endpoint'ы.
  - `admin/` — CRUD API для инструментов и шаблонов.
  - `runtime/` — пул инстансов агентов и управление сессиями.
  - `persistence/` — модели БД, репозитории и миграции.
  - `retrieval/` — поиск инструментов и агент-директорий.
  - `security/` — политики и фильтры доступов.
  - `observability/` — логи и метрики.
- `tests/` — автотесты (будут добавляться по мере реализации).
- `docker/` — инфраструктурные файлы для локального запуска.

## Быстрый старт
1. Установите зависимости:
   ```bash
   pip install -e .
   ```
2. Запустите проверки синтаксиса:
   ```bash
   python -m compileall src
   ```

Дальнейшая реализация следует этапам в `docs/03-implementation-plan-variant-b-v2.md`.
